# \<stripe-element\>

Element wrapper for the Stripe library

## Install

```
$ bower install stripe-element
```

## Use
```html
<stripe-element>
<!-- Your stripe-dependent code here -->
</stripe-element>
```

## Running Tests

```
$ polymer test
```
